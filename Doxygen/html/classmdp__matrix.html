Detailed Description

matrices of complex numbers

Example: 

///    mdp_matrix A,B;
///    A.dimension(3,3);
///    A(0,0)=A(1,1)=A(2,2)=A(1,2)=1.0+I/2;
///    B=A+inv(A)+exp(A+5);
///
