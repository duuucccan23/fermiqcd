Detailed Description

implements error propagation

Example: 

///    mdp_measure m;
///    // store 10 measurements
///    for(int i=0; i<10; i++) 
///       m << 3.0+mdp_random.gaussian(2.0);
///    m=sin(exp(m)+m);
///    cout << m.getmean() << "+/-" << m.geterr() << endl;
///

Assumes gaussian error propagation
